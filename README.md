# Encrypted Cloud Prescription Authentication System

A cloud-based system that verifies medical prescriptions using QR codes and secure cloud storage.

## Technologies Used

- AWS EC2
- AWS Lambda
- API Gateway
- Amazon S3
- QR Code Authentication
- HTML / CSS / JavaScript

## System Architecture

User uploads prescription → API Gateway → Lambda → EC2 → S3 Storage → Verification Response

## Features

• Secure prescription storage  
• QR-based authentication  
• Manual prescription ID verification  
• Cloud-based architecture  

## Note

AWS resources like EC2 and Lambda must be configured for the project to run.  
This repository contains project code and architecture.
